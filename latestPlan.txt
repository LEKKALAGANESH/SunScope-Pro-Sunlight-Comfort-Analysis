  Visual Realism vs Conceptual Clarity

  According to https://www.cityofsanmateo.org/DocumentCenter/View/10468/Shadow-Study-Guidelines:

  "Avoid excessive drawing detail which might make shadow lines hard to read."

  For early-stage analysis (your use case):
  - Conceptual clarity wins - Users need to understand sunlight patterns, not admire photorealistic materials
  - Massing-level geometry is appropriate and expected
  - Simple grey/neutral materials with differentiated building colors aid comprehension
  - Sharp, clean shadows are more valuable than soft, realistic penumbras

  Animated Sun-Path Videos

  According to https://sundiagram.com/, automated workflows for shadow analysis image sequences are highly valued.

  Recommended Format:
  - Duration: 15-30 seconds for full day
  - Frame rate: 24-30 fps for smoothness
  - Time display: Running clock overlay
  - Optional: Speed controls (1x, 2x, 4x)

  Elements to Include:
  - Moving sun disc along path arc
  - Time stamp (prominent, top-right)
  - Shadow transition (smooth interpolation)
  - Key metrics updating in real-time

  GLTF/3D File Exports

  Must Include:
  - Proper scene hierarchy (buildings as named groups)
  - Ground plane at correct position
  - Materials with meaningful names
  - Correct scale (meters)

  Optional Premium:
  - Sun position as a light source
  - Shadow-casting configuration
  - Camera presets (isometric, aerial, eye-level)

  Color & Shading Strategy for Sunlight Communication

  Building Colors (Current Implementation Analysis)

  Your current implementation uses random colors per building. Recommendation:

  Option A: Neutral Base + Highlight
  - All buildings: Light grey (#D1D5DB)
  - Selected/target building: Warm amber (#F59E0B)
  - Benefit: Focuses attention on analysis target

  Option B: Distinct but Muted Palette
  - Use 5-6 muted, distinguishable colors
  - Avoid pure red/green (colorblind accessibility)
  - Match legend exactly
